CSS Positions

        CSS Positions:
        There are five CSS positions: static, relative, absolute, fixed, and sticky.

        1. Static - The default position, where the element is not positioned and is placed in the normal document flow.

        2. Relative - The element is positioned relative to its normal position. Offset properties (top, bottom, left, right) are used to offset the element from its normal position.

        3. Absolute - The element is positioned relative to its nearest positioned ancestor (not the body element). Offset properties (top, bottom, left, right) are used to offset the element from its nearest positioned ancestor.

        4. Fixed - The element is positioned relative to the viewport of the browser. Offset properties (top, bottom, left, right) are used to offset the element from the viewport.

        5. Sticky - The element is positioned relative to its nearest positioned ancestor (not the body element), but the element is also "stuck" to the viewport when it is scrolled.

        The top, bottom, left, and right properties are used to offset the element from its normal position.

        Absolute depends on its ancestor for its position
