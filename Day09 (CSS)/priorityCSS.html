        Cascading Style Sheet(CSS) means it is like a waterfall,
        where if the style conflicts then the last one wins and it is applied

        Cascading Priority

        1. !important - no matter what, it will be applied
        2. Inline CSS
        3. Id Selector
        4. Class Selector
        5. Element/Tag Selector
        6. (If Tie)- Ordering is given priority to the last one/ recently added

        Priority Order: 

        !important > inline > id > class > tag > ordering
