        Overflow Property Values:

        visible (default): Makes overflowed content visible outside the element’s box, allowing it to overlap other elements.

        hidden: Clips overflowed content so it does not display outside the box; the extra content is completely invisible to the user.

        scroll: Always displays scrollbars (horizontal, vertical, or both), letting users scroll through any overflowed content—even if none exists.

        auto: Automatically adds scrollbars only when there is overflowing content; if the content fits, no scrollbars are shown.
