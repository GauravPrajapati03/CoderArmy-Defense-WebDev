CSS Units

px - one pixel is 1/96th of an inch but it depends on the device's screen size

em - relative to the font size of the parent element

rem - relative to the font size of the root element

vh - relative to the height of the viewport

vw - relative to the width of the viewport

Memory Usage

r - red (0 - 255) g - green (0 - 255) b - blue (0 - 255) a - alpha (0 - 1) 
    eg: rgba(255, 0, 0, 0.5)
    8 - bits (0 - 00000000, 255 - 11111111)
    Total size: 3 bytes - rgb(255, 0, 0) = 24 bits

    rgb(r, g, b) - 3bytes
    rgba(r, g, b, a) - 4bytes
    hsl(h, s, l) - 3bytes
    hsv(h, s, l, a) - 4bytes


    #hex - 3bytes - 0 - 9, A - F -> 16  eg  #008000 -> 00-red, green-80, blue-00

    #hexa - 4bytes - eg #00800080 -> 00-red, green-80, blue-00 alpha-80 (transparent)

rgb(r, g, b) - red, green, blue

rgba(r, g, b, a) - red, green, blue, alpha(transparent)

hsl(h, s, l) - hue, saturation, lightness

hsla(h, s, l, a) - hue, saturation, lightness, alpha(transparent)

hsv(h, s, v) - hue, saturation, value

hsva(h, s, v, a) - hue, saturation, value, alpha(transparent)
