Box Model

        Box model is a CSS layout mode that is used to create the layout of an HTML element. 
        It is a rectangular area that is used to contain the element's content and provide a border around it.

        The box model consists of four main parts:

        content - size of the content
        padding - area between the content and the border
        border - border of the element
        margin - area outside the border

        display: none - This property is used to hide an element completely from the user.
        It does not take any space in the layout.

        visibility: hidden - This property is used to hide an element completely from the user.
        It takes space in the layout.
