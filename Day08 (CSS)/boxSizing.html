Box Sizing

Box Sizing is a CSS property that determines how the width and height of an element are calculated. There are two types of box sizing: border-box and content-box.

  • Border Box: The width and height of an element include the content area and the padding and border areas.
    • width: content-width + padding-left + padding-right + border-left + border-right
    • height: content-height + padding-top + padding-bottom + border-top + border-bottom
  • Content Box: The width and height of an element only include the content area.
    • width: content-width
    • height: content-height

The box-sizing property can be used to switch between border-box and content-box. 

            .border-box {
                box-sizing: border-box;
            }

            .content-box {
                box-sizing: content-box;
            }
By default the box-sizing property is set to content-box but we change it to border-box in universal selector
