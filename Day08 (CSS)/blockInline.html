Block And Inline

        
        Block Elements - h1, div, p, ul, ol, table, form, section, article, header, footer, nav, aside, form, table

        Inline Elements - span, a, img, strong, em, small, b, i, input, br, label, select, iframe
Block Element Inline Element
Always starts on a new line Does not start on a new line
Take up full width available Takes up only content width/ only necessary width
Respects width and height Does not respect width and height
Respects top-bottom margin and padding (Only Horizontal) Partially respects margin and padding (only left and right)
Can be nested Cannot be nested
e.g., h1, div, p, ul, ol, table, form e.g., span, a, img
display: inline-block behaves both as block and inline
