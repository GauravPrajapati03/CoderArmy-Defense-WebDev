Lecture 3: Nested Lists and Tables

Nested lists in HTML allow placing one list inside another, creating a hierarchical or multi-level structure. This is useful for categorizing items or showing subtopics clearly.

Tables

Tables are used to display data in rows and columns, using the following tags:

Features:

  • border adds borders to the table and cells.
  • cellpadding adds padding inside each cell for readability.
  • cellspacing adds space between cells.
  • rowspan allows cells to span multiple rows.
  • colspan allows cells to span multiple columns.
  • align sets the alignment of the table.
  • bgcolor sets the background color of the table.
  • width sets the width of the table.

Table e.g.,

  • <table>: The container for the entire table.
  • <tr> (table row): Defines a row in the table.
  • <th> (table header): Defines header cells, usually bold and centered.
  • <td> (table data): Defines regular data cells.
  • <thead>: Defines the header section of the table.
  • <tbody>: Defines the body section of the table.
  • <tfoot>: Defines the footer section of the table.
