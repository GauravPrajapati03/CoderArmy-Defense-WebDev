Lecture 1: Introduction to Internet & Web Development

What is Internet?

The internet is a huge network that connects millions of computers and devices around the world, allowing people to quickly share information, send messages, and access websites from anywhere. It works like a giant web, linking together smaller networks so that anyone can find and use information, chat with others, watch videos, or even shop online. The internet makes it easy for people to communicate and find information in an instant, no matter where they are.
The Internet is a global network of interconnected computers and servers that communicate with each other using standardized protocols.
Communication is the process of exchanging information between computers and servers on the Internet.

What is Computer Network?

Computer networking is the way computers and other devices are connected together so they can share information, resources, and communicate with each other. It uses wires, cables, or wireless signals to link devices like computers, smartphones, printers, and more, making tasks like sending emails, sharing files, and browsing the internet possible. Networking lets many devices work together, keeps information easy to access, and supports things like chatting online, playing games, or working from different places.

Types of Networks

  • Personal Area Network (PAN): The smallest network type, designed to connect a few devices within a short range (typically 1 to 10 meters). Examples include Bluetooth connections between phones and wireless earbuds or infrared communication between a TV and remote.
  • Local Area Network (LAN): Connects computers within a limited area such as a home, office, or school building. It offers high-speed data transfer and is usually wired or Wi-Fi based. Examples include Wi-Fi networks in homes or offices.
  • Campus Area Network (CAN): Larger than a LAN but smaller than a city-wide network. It connects multiple LANs within a limited geographic area like a university campus or company buildings.
  • Metropolitan Area Network (MAN): Covers a larger geographic area like a city or town. MANs are used to connect several LANs within the city. Examples include cable TV networks or city-wide Wi-Fi networks.
  • Wide Area Network (WAN): Covers a very large area such as a country or multiple countries. The internet is the largest WAN. WANs use technologies like leased lines and satellite links.
  • Wireless Local Area Network (WLAN): A LAN that connects devices wirelessly using Wi-Fi technology, enabling mobility within the network area without cables.
  • Virtual Private Network (VPN): A secure network that uses public internet infrastructure to connect remote users or sites privately using encryption.

How Does Internet Works?

The internet works by quickly moving small pieces of data between computers through millions of connections

  • Data is split into small packets before sending so it moves efficiently.
  • Routers and switches guide those packets from one computer to another along the fastest path.
  • Packets travel using special rules called protocols (like TCP/IP) to make sure they arrive correctly.
  • Web browsers and apps reassemble packets so people see complete websites, images, or messages.
  • Everything travels through wires, cables, or wireless signals to connect computers and devices worldwide.

What is IP Address?

An IP address is a unique number assigned to every device connected to a network so it can be identified and can send or receive data.

Types and Versions

  • IPv4: Uses 32 bits, appearing as four numbers separated by dots (e.g., 192.168.1.1). Supports over 4 billion addresses but has limitations as more devices connect to the internet.
  • IPv6: Uses 128 bits, written in eight groups of four hex digits separated by colons (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334). Supports a vast number of devices, solving the address shortage problem.

Address Types

  • Public IP address: Assigned by an Internet Service Provider (ISP) and visible on the internet. Used to identify networks online. These can be reached from anywhere on the internet.
  • Private IP address: Used within a local area network such as a home or office. Not visible from outside the internal network. Examples include 192.168.x.x, 10.x.x.x, and 172.16.x.x ranges.
  • Static IP address: Does not change; manually assigned and useful for servers and other devices that need consistent identification.
  • Dynamic IP address: Changes automatically, usually assigned by a DHCP server in private networks or by ISPs on the internet.

Addressing Methods

  • Unicast: One-to-one communication between two devices. Each message goes from a unique sender to a unique receiver, which is the most common addressing method.
  • Broadcast: Delivering one packet to all devices in a local network at once—possible only in IPv4 (typical address 255.255.255.255).
  • Multicast: Sends packets from one sender to multiple specific receivers participating in a multicast group (range 224.0.0.0 to 239.255.255.255).
  • Anycast: Sends a packet to the nearest one in a group of receivers, enabling load balancing and minimizing latency for services like DNS.

What is DNS?

DNS (Domain Name System) is a system that translates human-readable domain names into IP addresses that computers can understand.

It is like the phonebook of the internet that translates human-friendly website names (domain names) into IP addresses computers use to find those websites.

How Does DNS Works?

  1. When a user types a website address (like www.example.com) in a browser, the browser first checks its local cache to see if it already knows the IP address.
  2. If not found, the request goes to a DNS resolver (usually provided by the Internet Service Provider).
  3. The DNS resolver asks a root DNS server which directs it to the Top-Level Domain (TLD) server (like .com or .org).
  4. The TLD server points the resolver to the authoritative DNS server responsible for the domain (example.com).
  5. The authoritative server returns the IP address of the website.
  6. The resolver sends the IP address to the user's computer, allowing the browser to connect to the website's server and load the page.

Key Points

  • DNS eliminates the need to remember complex IP addresses by using easy-to-remember domain names.
  • It is hierarchical and distributed worldwide to handle the vast number of internet domains, making the system scalable and fast.
  • DNS plays a crucial role in web browsing, email delivery, and many other internet services by resolving names to addresses quickly.
  • This entire DNS lookup process happens in milliseconds, making internet browsing smooth and efficient.

What is MAC Address?

A MAC address (Media Access Control address) is a unique hardware identifier assigned to a network interface controller (NIC) of a device by its manufacturer. It works at the Data Link Layer (Layer 2) of the OSI model and is used to identify devices within a local network.

  • It is a 48-bit address usually expressed as six groups of two hexadecimal digits separated by colons or hyphens (e.g., 00:1A:2B:3C:4D:5E).
  • The first half identifies the manufacturer (called Organizationally Unique Identifier - OUI), and the second half identifies the specific device.
  • Every network interface (Wi-Fi card, Ethernet port) has its own unique MAC address.
  • MAC addresses are permanent in hardware but can sometimes be changed or "spoofed" through software.
  • Used for communication within a local area network (LAN) to ensure data goes to the correct device.
  • MAC address is also known as the physical or hardware address, distinguishing it from IP addresses, which are logical network addresses.

Address Resolution Protocol (ARP)

Address Resolution Protocol (ARP) is a network protocol used to map an IP address to a MAC (hardware) address, allowing devices within a local area network (LAN) to communicate with each other.

What are Ports?

A port in computer networking is a virtual communication endpoint used by devices to differentiate between multiple services or applications running on the same device over a network. Each port is identified by a unique port number, allowing a computer to manage many simultaneous connections through a single IP address

  • Ports are software-based and managed by the operating system.
  • Port numbers range from 0 to 65535 (16-bit number).
  • Ports enable routing of data to the correct applications or services on a device
  • Working alongside IP addresses, ports help direct network traffic precisely like telephone extensions in a phone system.

Categories of Ports

  • Well-Known Ports (0-1023): Reserved for common, widely-used services and protocols (e.g., HTTP, FTP).
  • Registered Ports (1024-49151): Assigned to specific services by IANA but not as commonly used.
  • Dynamic or Private Ports (49152-65535): Used for temporary or private communications, often by client applications.

Web Development

Web development is the process of creating and maintaining websites and web applications, involving both how a site looks and how it works behind the scenes.

Front-End Development

Focuses on the client side, what users see and interact with in their web browsers.

Back-End Development

Focuses on the server side, how data is handled, stored, and served to the front end.

It typically involves three core technologies:

  • HTML (HyperText Markup Language): The basic structure and content of web pages.
  • CSS (Cascading Style Sheets): Styling and layout of web pages to make them visually appealing.
  • JavaScript: Programming language to make web pages interactive and dynamic, adding elements like forms, animations, and real-time updates.
