Lecture 6: HTML5 New Tags and Attributes

Topics Covered:

  • Audio Tag
  • Video Tag
  • Iframe Tag

  1. <audio> Tag

    The <audio> tag is used to embed sound content such as music or other audio streams directly on a web page.

    Example:

    
<audio source src="audio-file.mp3" 
type="audio/mpeg" controls> </audio>
                
                or

<audio controls>
  <source src="audio-file.mp3" type="audio/mpeg">
  <source src="audio-file.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>


    • The controls attribute displays playback controls (play, pause, volume).
    • The <source> tag allows multiple formats for browser compatibility.
    • Common audio formats: MP3, OGG, WAV.
    • Other attributes include autoplay, loop, muted, and preload.

  2. <video> Tag

    The <video> tag embeds video content such as movie clips or video streams.

    Basic Example:

    
<video source src="video-file.mp3" 
type="audio/mp4" controls> </video>

                or

<video width="640" height="360" controls>
  <source src="video-file.mp4" type="video/mp4">
  <source src="video-file.webm" type="video/webm">
  Your browser does not support the video tag.
</video>


    • The controls attribute provides play, pause, and volume controls.
    • Attributes include autoplay, loop, muted, poster (image shown before playback), width, and height.
    • Supports captions through <track> elements.

  3. <iframe> Tag

    The <iframe> tag embeds another HTML document inside the current page, often used for embedding videos (e.g., YouTube), maps, or other external webpages.

    Basic Example (Embedding a YouTube Video):

    
<iframe width="560" height="315" 
src="https://www.youtube.com/embed/VIDEO_ID" 
frameborder="0" allowfullscreen>
</iframe>
    • src: URL of the embedded content.
    • width and height control the size.
    • allowfullscreen enables full screen.
    • frameborder controls the border visibility.
    • Can customize allowed features with allow attribute.
    • We can also display docs, pdfs, other webpages.
