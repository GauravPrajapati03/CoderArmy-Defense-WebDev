Lecture 5: Forms in HTML

An HTML form is a key element used to collect user input and send it to a server for processing. It consists of various input controls where users enter data, such as text fields, checkboxes, buttons, and more.

Input Types & Attributes

HTML <input> element supports many types to collect various kinds of user data.

Common Input Types:

Input Type Description Example Code
text Single-line text input <input type="text" name="username">
password Password input (obscured text) <input type="password" name="pass">
email Email input with validation <input type="email" name="email">
number Number input with min/max options <input type="number" name="age" min="1" max="99">
tel Telephone number <input type="tel" name="phone">
url URL input type <input type="url" name="website">
search Search box <input type="search" name="query">
checkbox Checkbox selection (multiple options) <input type="checkbox" name="subscribe">
radio Radio buttons (single choice) <input type="radio" name="gender" value="M">
file File select dialog <input type="file" name="resume">
hidden Hidden input (not visible, sent with form) <input type="hidden" name="userId" value="123">
button Generic clickable button <input type="button" value="Click Me">
submit Submits the form <input type="submit" value="Submit">
reset Resets the form <input type="reset" value="Reset">
color Color picker <input type="color" name="favcolor">
date Date picker <input type="date" name="dob">
datetime-local Date and time picker (no time zone) <input type="datetime-local" name="appointment">
month Month and year picker <input type="month" name="expdate">
week Week picker <input type="week" name="week">
time Time picker <input type="time" name="alarm">
range Slider to select a value within a range <input type="range" name="volume" min="0" max="10">
image Image submit button <input type="image" src="submit.png" alt="Submit">

Common Input Attributes:

  • name: Name used to identify the input element during form submission.
  • id: Unique identifier for the input (useful for labels and scripting).
  • value: Specifies the default or current value of the input.
  • placeholder: Text shown when input is empty to guide users.
  • required: Specifies that the input must be filled before submitting.
  • disabled: Disables the input so it cannot be interacted with.
  • readonly: Makes the input read-only, not editable by the user.
  • min / max: Define minimum and maximum values (for number, date, etc.).
  • step: Defines legal intervals for numeric or date/time inputs.
  • pattern: Regex pattern for input value validation.
  • autocomplete: Suggests previously entered values while typing.
  • multiple: Allows selecting multiple files (for <input type="file">).
