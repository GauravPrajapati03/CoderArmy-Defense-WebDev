Lecture 2: What is HTML & it's Basic Tags

What is HTML?

HTML stands for HyperText Markup Language. It is the core language used to create the structure and content of web pages on the internet. It allows you to structure and format text, images, links, and other elements on a web page. HTML is a markup language, which means it uses special tags to define different elements and their attributes.

Basic HTML Tags

Here are some basic HTML tags you should know:

  1. Heading Tags

    HTML heading tags are used to define headings and subheadings on a web page, helping to organize content with a clear structure. There are six levels of heading tags, from <h1> to <h6>, with <h1> representing the most important heading and <h6> the least important.

    • <h1> - Heading 1
    • <h2> - Heading 2
    • <h3> - Heading 3
    • <h4> - Heading 4
    • <h5> - Heading 5
    • <h6> - Heading 6
    e.g., <h1>Hello World!</h1>
                                
    Main Heading
    
                            
    Section Title
    
                            
    Subsection Title
    
                            
    Small Subsection
    
                            
    Minor Section
    
                            
    Smallest Section

  2. Paragraph Tag

    The <p> tag is used to define a paragraph of text on a web page. It is a block-level element, which means it takes up the full width of the page and creates a new line.

    e.g., <p>This is a paragraph of text.</p>

  3. Horizontal Rule Tag

    The <hr> tag is used to create a horizontal line on a web page, often used to separate sections of content.

    e.g., <hr> or <hr />

  4. Line Break Tag

    The <br> tag is used to insert a line break in a paragraph, creating a new line.

    e.g.,
                        
    This is line one.<br>This is line two.

  5. Lists

    HTML provides several types of lists, including unordered lists (<ul>), ordered lists (<ol>), and nested lists.

    • <ul> - Unordered List - Bullet Points
    • <ol> - Ordered List - Numbers, Letters
    • <dl> - Definition List - Description Lists
    • <dt> - Definition Term
    • <dd> - Definition Description
    • <li> - List Item

  6. Links - Anchor Tag

    The <a> tag is used to create hyperlinks on a web page. It allows you to link to other pages or resources on the internet.

    e.g., <a href="https://www.example.com">Example
                     Link</a>

  7. Image Tag

    The <img> tag is used to embed images on a web page. It allows you to display images on a web page.

    e.g., <img src="image.jpg"
alt="Image Description" width="300"
 height="200">
